# Audit Log Service

##  Overview
Audit Log Service — backend-сервис для централизованного хранения и поиска audit-событий,
фиксирующих действия пользователей и систем в корпоративных приложениях.

Сервис используется для:
- аудита действий пользователей
- расследования инцидентов
- соответствия требованиям безопасности и compliance
- аналитики и мониторинга

---

## Business Context
Проект разработан по заказу B2B-платформы с несколькими backend-сервисами и административной панелью.

Цель — создать единый сервис, который принимает audit-события от разных систем и предоставляет API для их поиска и анализа.

---

##  Features
- регистрация audit-событий
- хранение истории действий
- поиск событий по пользователю
- пагинация и сортировка
- поддержка различных типов действий и источников
- централизованная обработка ошибок
